Description
===============

* Soap WebService of users
* Endpoint
    * Create user
    * Update user
    * Get user info by user email

Stack
========

* JDK1.7
* Spring MVC 4 & mybatis
    * Spring WS 2.3 (SOAP Web Service)
    * mybatis3 (Persistance)
* Oracle Database 12c EE 12.2.0.1.0 64bit
* WildFly9

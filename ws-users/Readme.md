Description
===============

* Soap WebService of users
* Endpoint
    * Create user
    * Update user
    * Get user info
    * Login user

Stack
========

* JDK1.7
* Spring MVC 4 (SOAP Web Service)
* JAVA EE7
    * JAAS (Authentication and Authorization)
    * mybatis3 (Persistance)
* Oracle Database 12c EE 12.2.0.1.0 64bit
* WildFly9
